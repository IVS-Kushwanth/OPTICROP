# OptiCrop — Agricultural Crop Recommendation System

> **AI-ML and GEN-AI Track** | 2026–2027

## Project Overview

OptiCrop is a machine learning-powered crop recommendation system that accepts seven soil and
climate parameters and recommends the optimal crop to plant. Built with Python (scikit-learn + Flask)
and delivered through a responsive Bootstrap 5 web interface.
